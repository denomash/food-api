  6. User login

Getting started

In order to use the api you must have the following in mind.

  1. Posting an order

    2. To post an order we shall use the following endpoint:

    • POST at endpoint /api/v1/orders

      • The payload should have:

      • item
      • price
      • quantity
      • address

  2. Getting all orders

    3. To get all orders we shall use the following endpoint:

    • GET at endpoint /api/v1/orders

  3. Getting an order by id

    4. To get an order by id we shall use the following endpoint:

    • GET at endpoint /api/v1/orders

      • Use a valid id or else you will get an error response


  4. Updating an order

    5. To update an order we shall use the following endpoint:

    • PUT at endpoint /api/v1/orders

      • The payload should have a status that is either 'pending' or 'completed'


  5. User signup

    6. To signup a user we shall use the following endpoint:

    • POST at endpoint /api/v1/signup

      • The payload should have:

      • username
      • email
      • password
      • confirm password

  6. User login

    7. To login a user we shall use the following endpoint:

    • POST at endpoint /api/v1/signup

      • The payload should have:

      • email
      • password
